COVID Ipsum

"Covid Ipsum" is a program for creating dummy text. Unlike the traditional “Lorem Ipsum”, this text is all centred around the theme of Covid 19. A user can generate a series of totally random paragraphs with the click of a button. Although the resulting sentences often makes little sense they’re usually good for a laugh. To have a go, click here.

Same Same

“Same Same” was created to help children with ASD. It is a simple game in which a player is presented with four images and has to identify which images , if any, share the same category. Matching skills are an essential part of language development for children with autism and this game provides a fun way to strengthen these skills. Click here to play.

100 Horrors

Best viewed on a large screen device, “One Hundred Horrors” is an interactive art installation. Click on any of the 100 televisions for some terrifying sights and sounds. Feeling brave? Click here!

Astronauts and Holy Crosses

“Astronauts and Holy Crosses” is a simple Tic-Tac-Toe game created using HTML, CSS and JavaScript. Science is represented by an astronaut and religion is represented by a cross. Pick a side and click here.

Magic Melbourne Moments

“Magic Melbourne Moments” was a collaborative project I contributed to as part of an Immersive Software Engineering Course at General Assembly Melbourne. The concept is a website that allows a tourist visiting Melbourne to define their interests and be presented with a list of relevant recommendations. The project was built in node.js and pulled data from a number of APIs including Sygic Map, Mapbox and OpenWeatherMap. For more information, click here.
